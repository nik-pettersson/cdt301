Program-> Func Program | Func
Func-> F_TYPE ID ( Params ) { F_Body }
F_TYPE-> VOID | INT
Params-> VOID | Param
Param-> Param COMMA P_Decl | P_Decl
P_Decl-> INT ID
F_Body->  Declerations Line
Declerations-> Decl Declerations | Decl
Decl-> INT ID SEMI
[LOOPS]
LoopBody-> ( Loop_Statement ) { Line } | ( Loop_Statement ) Statement ; 
If-> IF LoopBody Else
Else-> { Statement } | Statement | e
While-> WHILE LoopBody
Loop_Statement-> Relationop | Statement
[CODE]
Line-> L_Statement ; Line | L_Statement ; | Assignop ; | If | While
L_Statement-> Read | Write | Statement
Statement-> SubAdd | Notop | MulDiv  
Return-> Return | Return ( Statement ) | Return ID

SubAdd-> Statement - Term | Statement + Term
MulDiv-> Term * Statement | Terms / Statement
Term = ID | NUM

Read-> READ ID
Write-> WRITE Statement | Write Term
Notop-> NOTOP ID | NOTOP ( Statements )

Assignop-> ID = Statement

Relationop-> Statement RELATIONOP Statement | Statement RELEATIONOP Term | Term RELATIONOP Statement | Term RELATIONOP Term
